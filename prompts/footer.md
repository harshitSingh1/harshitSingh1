# Footer — Design

Mirrors the hero. Closes the engineering document.

---

## Design Goal

Bookend the profile with the same blueprint spec plate language. No generic sign-offs.

---

## Rejected patterns

- "Thanks for visiting!"
- Motivational quotes
- "Made with ❤️"
- Visitor counters
- Social icon rows
- Marquee text

---

## Selected Design — Document Closure Plate

**Implementation:** `assets/footer-spec.svg`

### Structure (mirrors hero)

| Hero field | Footer equivalent |
|------------|-------------------|
| System title | `DOCUMENT CLOSURE` |
| Operator | `CONTACT` endpoints |
| Domains | — (omitted — already stated) |
| Verified metrics | `STATUS: ACTIVE — ACCEPTING COLLABORATION` |
| Links | GitHub · LinkedIn |
| Title block | `SHEET 01/01 — END OF SPEC` |
| Document ID | `HS-ENG-2024 · Rev 3.0 · Greater Noida, IN` |

### Visual elements (shared with hero)

- Blueprint grid background
- Registration marks (four corners)
- 1px `#e5e5e5` frame border
- Mono labels in `#a3a3a3`
- Accent `#3b6fa8` for links and marks
- Title block rectangle bottom-right
